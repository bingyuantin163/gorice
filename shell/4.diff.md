### 基本命令

``` diff
# 字母”a”、”d”、”c”分别表示添加、删除及修改操作
diff file1 file2 # 直接比较

# n1 a n3,n4
# n1,n2 d n3
# n1,n2 c n3,n4
# ”n1”、”n2”表示在文件1中的行号，”n3”、”n4”表示在文件2中的行号。
# "|"表示前后2个文件内容有不同
# "<"表示后面文件比前面文件少了1行内容
# ">"表示后面文件比前面文件多了1行内容

diff log2014.log log2013.log  -y -W 50
# 2013-01                 2013-01
# 2013-02                 2013-02
# 2014-03               | 2013-03
# 2013-09                 2013-09
# 2013-10                 2013-10
# 2013-11               <
# 2013-12               <
```
